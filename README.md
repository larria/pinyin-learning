# 快乐学拼音 - 幼儿启蒙 (Happy Pinyin Learning)

这是一个专为幼儿设计的拼音学习 Web 应用程序。通过可爱的界面、生动的 Emoji 和互动发音，帮助孩子轻松掌握汉语拼音。

## ✨ 功能特色

*   **三大分类学习**：
    *   **声母 (Initials)**：包含 23 个声母的学习卡片。
    *   **韵母 (Finals)**：包含 24 个韵母的学习卡片。
    *   **整体认读音节 (Overall Syllables)**：包含 16 个整体认读音节。
*   **互动闪卡**：点击卡片即可听到标准发音（使用 Web Speech API）。
*   **图文并茂**：每个拼音都配有相关的 Emoji 和常用词汇（如：a - 🍎 苹果），加深记忆。
*   **可爱界面**：
    *   充满童趣的配色和字体（马善政毛笔字体、站酷快乐体）。
    *   "家庭小分队"（猪爸爸、鱼妈妈）陪伴学习，提供鼓励语。
*   **简单易用**：大按钮设计，方便儿童操作；支持左右切换卡片。

## 🚀 如何运行

本项目是一个纯静态网页应用，无需安装复杂的依赖或构建工具。

1.  **克隆或下载**本项目到本地。
2.  直接双击打开 `index.html` 文件即可在浏览器中运行。
3.  或者，如果你使用 VS Code，可以安装 "Live Server" 插件，右键点击 `index.html` 选择 "Open with Live Server" 以获得更好的体验。

## 📂 文件结构

```
.
├── index.html      # 主页面结构
├── style.css       # 样式文件 (CSS3, 响应式设计)
├── script.js       # 主要逻辑 (交互, 音频播放, 路由)
├── data.js         # 拼音数据 (包含字符, Emoji, 词汇, 发音文字)
└── README.md       # 项目说明文档
```

## 🛠️ 技术栈

*   **HTML5**: 语义化标签构建页面结构。
*   **CSS3**: Flexbox 布局，CSS 变量，动画效果。
*   **JavaScript (ES6+)**: 模块化开发 (`type="module"`), DOM 操作。
*   **Web Speech API**: 利用浏览器原生能力实现语音朗读功能。

## 📝 注意事项

*   语音朗读功能依赖于浏览器的 Web Speech API 支持。建议使用最新版本的 **Google Chrome** 或 **Edge** 浏览器以获得最佳体验。
*   部分浏览器可能需要网络连接才能正常合成语音。
